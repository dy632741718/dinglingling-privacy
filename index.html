叮铃铃 隐私政策

最后更新日期：2026年5月12日

叮铃铃（以下简称"本应用"）高度重视您的隐私。本隐私政策旨在向您说明本应用如何收集、使用和保护您的个人信息。在您同意本隐私政策前，本应用不会初始化任何第三方 SDK 或收集任何设备信息。

1. 信息收集

本应用不收集以下信息至任何远程服务器：

  • 短信内容
  • 联系人信息
  • 通话记录
  • 位置信息
  • 任何个人身份信息

所有短信匹配和处理均在您的设备本地完成。

2. 第三方 SDK 信息

本应用集成了以下第三方 SDK：

SDK 名称友盟+ SDK
服务类型数据统计分析、广告投放
收集个人信息类型设备信息（IMEI / Mac / Android ID / IDFA / OPENUDID / GUID / SIM卡 IMSI / 地理位置信息）
隐私权政策链接https://www.umeng.com/page/policy

友盟 SDK 收集的上述设备信息仅用于数据统计分析和广告投放优化，不会与本应用的短信内容产生任何关联。

3. 权限使用说明

  • 读取短信：用于实时识别用户设定的关键字，匹配时触发警报。短信数据仅本地处理，不上传。
  • 弹窗显示：用于在锁屏或其他应用上方显示全屏警报提醒。
  • 前台服务：用于在后台持续运行短信监听服务，确保不被系统关闭。
  • 通知权限：用于发送系统通知栏提醒。
  • 振动：用于警报触发时提供触觉反馈。

4. 数据安全

所有短信匹配规则、历史触发记录等数据均仅存储在您的设备本地。本应用不会将任何用户数据上传至服务器或与第三方共享（友盟 SDK 的设备统计数据除外，且该数据不与短信内容关联）。

5. 您的权利

您可以随时在系统设置中撤销本应用的任何权限。首次启动时，本应用会在您同意隐私政策后才初始化第三方 SDK。
