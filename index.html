叮铃铃 隐私政策

最后更新日期：2026年5月12日

叮铃铃（以下简称"本应用"）高度重视您的隐私。本隐私政策旨在向您说明本应用如何收集、使用和保护您的个人信息。

1. 信息收集

本应用不收集以下信息：

  • 短信内容
  • 联系人信息
  • 通话记录
  • 位置信息
  • 任何个人身份信息

所有短信处理均在您的设备本地完成，不会上传至任何远程服务器。

2. 第三方 SDK 数据收集

本应用集成了友盟+ SDK（com.umeng.umsdk），用于数据统计和广告投放。友盟 SDK 可能会收集以下设备信息：

  • 设备标识符（IMEI / MAC / Android ID / IDFA / OAID）
  • 设备型号、操作系统版本
  • 网络信息（IP 地址、网络类型）
  • 应用使用情况（如广告展示和点击数据）

友盟 SDK 收集的上述信息仅用于优化广告投放和数据统计分析，不会与您的短信内容关联。友盟的隐私政策详见：https://www.umeng.com/policy

3. 权限使用说明

  • 读取短信：用于实时识别用户设定的关键字，在匹配时触发警报提醒。短信数据仅本地处理。
  • 弹窗显示：用于在锁屏或其他应用上方显示全屏警报，确保用户及时收到重要短信提醒。
  • 前台服务：用于在后台持续运行短信监听服务，确保监控不被系统关闭。
  • 通知权限：用于发送系统通知栏提醒，即使在锁屏状态下也能接收警报。
  • 振动：用于在警报触发时提供触觉反馈。

4. 数据安全

所有短信匹配规则、历史触发记录等数据均仅存储在您的设备本地。本应用不会将任何用户数据上传至服务器或与第三方共享（友盟 SDK 的设备统计数据除外）。

5. 您的权利

您可以随时在系统设置中撤销本应用的任何权限。撤销后，相关功能将无法正常使用。

如需删除友盟 SDK 收集的设备数据，您可以在本应用的"设置"中重置广告标识符，或联系友盟官方。
